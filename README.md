## 概要
個人営業で施術人数が限られているトレンド×ニュアンス特化を目的とした、プライベートネイルサロン予約専用ランディングページ（LP）**です。
SNSからの顧客拡充を目的とし、ファッションや美容への感度が高い層をターゲットに予約への導線設計とブランドの世界観を両立させたデザイン・実装を行いました。

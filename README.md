Gardenia（nailsalon-B）

## 概要
トレンド感のあるニュアンスネイルに特化した、「感度の高い層」向けのプライベートサロン予約専用LPです。
SNS（Instagram等）での発信を資産とし、フォロワーをスムーズに予約へと繋げるための「ブランドの世界観の受け皿」として設計・実装しました。

- レスポンシブ設計: スマートフォンでの閲覧・予約操作をメインに想定したモバイルファースト設計
- インタラクション:
- 直感的に操作できる予約フォームへの誘導ボタン

## 制作意図（課題解決アプローチ）
1. SNSの世界観を拡張するエディトリアル・デザイン
- ターゲット: 20代〜30代前半のアパレル・クリエイティブ職など、自己表現を重視する女性。
- 戦略: ターゲットが日常的に触れているファッション誌のようなグリッドレイアウトを採用。落ち着いた「リップカラー」をイメージした配色で、大人の色気とトレンド感を両立させ、SNSの画像だけでは伝えきれない「サロンの空気感」を具現化しました。
2. 直感的な「センスへの信頼」を醸成する構成
- 課題: ニュアンスネイルは言語化しにくい「センス」が商品。
- 解決策: 説明的な文章を削ぎ落とし、タイポグラフィと余白のバランスで「このサロンならお任せできる」という直感的な信頼を構築。あえてグリッドを崩した配置にすることで、型にハマらないデザインの柔軟性を表現しました。
3. スムーズなコンバージョン導線の確保
- 解決策: SNSで「可愛い」と感じた熱量を逃さず、空き状況の確認から予約までを最短で行えるよう、モバイルファーストでのボタン配置とシンプルな画面遷移を徹底しました。

---

## 工夫した点
- 非対称グリッドレイアウト: CSS Gridを活用し、雑誌のコラージュのような動きのあるレイアウトをレスポンシブで実現。
- 視覚的な質感表現: ニュアンスネイルの繊細な色彩を活かすため、背景色や文字色のコントラストを微調整し、写真が最も映えるトーンを追求。
- 軽量化と高速化: 画像点数が多い構成ながらも、次世代フォーマットの検討や最適化を行い、ストレスのない閲覧環境を構築。
